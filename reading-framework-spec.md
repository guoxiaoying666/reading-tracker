# 儿童阅读成长体系设计文档

**版本：** v1.6  
**适用对象：** 10岁，女，文学人文基础强，自然科学待补  
**文档用途：** 供 Claude Code 引用，构建 HTML 阅读记录与诊断建议系统

-----

## 一、设计思路

### 1.1 核心目标

三个层级目标，优先级依次递进：

1. **补齐阅读广度**：解决当前阅读结构偏窄问题，系统性覆盖七大知识维度
1. **支撑全面发展**：以广义阅读（书籍、纪录片、音频、优质短视频）为媒介，覆盖阅读能触达的发展维度
1. **发展个人专长**：在孩子兴趣集中的领域，从广度走向深度

### 1.2 设计原则

- **知识积累为主轴**，能力培养为副轴（能力标注作参考，不驱动选书）
- **七个维度并行**，无绝对优先级，有比重差异
- **哲学打散嵌入**：不单独成维度，以思维方式形态嵌入每个维度
- **三级统一按题材分**：地域（中文/外文）、形式（名著/虚构/非虚构）作为独立属性字段，不进入分类体系
- **归类按主要观点表达**：形式是次要的，内容表达是归类主要依据
- **体系动态更新**：录书过程即是完善分类的过程，遇到空缺及时补充

## 二、理论来源说明

### 2.1 框架结构：IB MYP（国际文凭中学项目）

**机构：** 国际文凭组织（IBO）｜**适用年龄：** 11–16岁  
**采纳内容：** 维度均衡分布逻辑、知识论贯穿各学科的方式  
**未直接用的原因：** IB是学校课程闭环，缺”自我认知”独立维度，中国文明比重不足

### 2.2 一级维度逻辑：以人的发展为骨架

以”孩子认识世界的视角”构建维度，参照 IB MYP 全球背景框架 + 皮亚杰认知发展理论

### 2.3 二级学科划分：GB/T 13745-2009

**标准：** 中国学科分类国家标准｜**机构：** 国家技术监督局

### 2.4 中文根基：义务教育课程方案（2022版）

**机构：** 中华人民共和国教育部  
**采纳内容：** 中国史、语文、艺术维度的内容深度；各科课时比重参考

### 2.5 能力对应：北师大5C模型（2018）

**机构：** 北京师范大学中国教育创新研究院 × 美国P21联盟  
**5C：** 文化理解 / 审辨思维 / 创新 / 沟通 / 合作  
**使用方式：** 副轴标注，不驱动选书

### 2.6 文学分类参考：丹妈读童书 + 小花生网

**丹妈：** 题材/体裁分法参考（童话寓言、幻想小说、推理小说、动物小说、传记散文等）  
**小花生：** 虚构/非虚构二分法作为书籍属性字段参考  
**使用方式：** 文学二三级拆解参考，不作为框架骨架依据

### 2.7 比重依据：开卷图书零售市场数据（2023）

**来源：** 北京开卷信息技术有限公司年度报告  
**采纳内容：** 少儿科普百科27%、少儿文学20%、少儿绘本20%为三大品类，据此推导各维度目标中值及文学/艺术内部二级占比


## 六、诊断建议系统逻辑

### 6.1 覆盖度诊断

每个维度以5本/年为下限，目标中值百分比为参考：

- 未满5本 → 🔴 需补充
- 满5本且未超目标中值 → 🟢 健康
- 超过目标中值明显 → 🟡 注意偏科

### 6.2 结构均衡诊断

- 虚构/非虚构比例（10岁目标参考：虚构60% / 非虚构40%）
- 中文/外文比例（目标参考：中文50% / 外文50%）
- 名著占比追踪

### 6.3 三级盲区检测

在已覆盖维度内，检测哪些三级节点从未涉及，提示需补充方向。

### 6.4 建议输出格式

```
本月已读 X 本
虚构 X 本 / 非虚构 X 本｜中文 X 本 / 外文 X 本

维度覆盖：
🔴 未满5本：②社会人文 / ③自然 / ④历史文明 / ⑥艺术 / ⑦科技创新
🟡 偏科：⑤文学（超出目标中值）
🟢 健康：①自我

下月建议方向：
→ ③自然：生物生命 / 天文宇宙
→ ④历史文明：中国史
→ ②社会人文：人际沟通
→ 增加非虚构比重
```

### 6.5 触发时机

- 每新增一本实时更新
- 每月末生成月度报告
- 累计满20本生成阶段诊断

## 七、给 Claude Code 的构建说明

### 7.1 整体结构

HTML 单文件应用，三个主 Tab：

1. **书架**（首页）：可视化统计 + 书目列表
1. **记录**：新增书籍表单
1. **规划**：体系框架说明 + 诊断建议

### 7.2 首页可视化

- **七维度雷达图**：实际比重 vs 目标中值
- **结构饼图**：虚构/非虚构 + 中文/外文 两组
- **月度柱状图**：12个月横向对比
- **维度气泡**：点击筛选对应书目
- **书卡列表**：可按维度/月份/语言/虚构/评分筛选排序

### 7.3 记录表单字段（按填写顺序）

1. 书名（必填）
1. 作者
1. 读完月份
1. 来源（广州图书馆 / 自购 / 学校图书馆 / 其他）
1. 语言（中文 / 外文）
1. 形态（虚构 / 非虚构）
1. 名著（是 / 否）
1. 一级维度（下拉）
1. 二级分类（联动下拉）
1. 三级节点（联动下拉，诗歌除外）
1. 家长评分（1–5星）
1. 孩子评分（1–5星，视觉突出，让孩子自己点）
1. 一句话感受

### 7.4 诊断建议面板

- 实时显示各维度缺口
- 显示虚构/非虚构、中文/外文比例偏差
- 基于三级盲区给出补充方向标签
- 不硬编码书单，只输出方向

### 7.5 数据持久化

localStorage 存储，JSON格式，支持导出。

### 7.6 设计风格

- 移动端优先
- 暖色系，适合亲子场景
- 孩子评分区域视觉大、醒目、有趣
